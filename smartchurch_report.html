The Work

What We Built Today

Nine features moved from placeholder to production — plus one bonus. Here's the before & after for each.

1🔔 Notification System
Before
Settings toggle existed but no actual notifications were sent.
→
After
Real notifications fire for challenges, new chats, and match requests.
2🔄 Matching & Events Real-time Sync
Before
Data was only visible on the user's own device.
→
After
Firestore integration — all members see the same live data.
3✅ To-do ↔ Challenge Auto-check
Before
To-dos and challenges were completely separate.
→
After
Completing a flagged 🚩 to-do automatically checks the linked challenge.
4🎂 Birthdate Field on Sign-up
Before
Only name was required at registration.
→
After
Birthdate is now collected and used for age-gated features.
5📖 60 Bible Verses
Before
Only 7 verses — users saw repeats constantly.
→
After
30 OT + 30 NT verses, rotating by day of the year.
6💬 Auto DM Room on Match Accept
Before
Users had to manually create a chat room after matching.
→
After
A dedicated DM room is instantly created with a welcome message.
7📲 PWA — Home Screen Install
Before
Users had to open a browser every time.
→
After
The app can be installed as an icon directly on the phone home screen.
8💑 Romantic Matching Feature
Before
Showed a "Coming Soon" placeholder only.
→
After
Full matching UI with a 19+ age-verification gate.
9🔔 Browser Push Notifications
Before
Notifications were only visible inside the app.
→
After
Real browser / device notification popups are supported.
🎁Bonus — Event Ticket Cross-device Sync
Before
Tickets booked on Phone A were invisible on Phone B.
→
After
Firestore sync — reservations are visible on any device.
